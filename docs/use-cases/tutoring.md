# One-to-One Tutoring

DrawShare works naturally for private tutoring: the tutor draws and explains while the student watches the canvas update live. After the session, export each page as a PNG so the student has a visual record to review.

## Setup

**Scenario A — Same Machine**

If tutor and student are sitting at the same computer (e.g. screen-sharing over a video call), open DrawShare in two browser windows. Start a session in one and join in the other. Both windows stay in sync.

**Scenario B — Same Network**

If tutor and student are on separate devices connected to the same Wi-Fi:

1. Tutor opens DrawShare and clicks **Share**. Note the session code.
2. Student opens DrawShare on their device, clicks **Join Session**, and enters the code.
3. The student's view mirrors the tutor's canvas in real time.

## During the Session

- Create a **new page** for each topic or problem set. Student view follows page switches automatically.
- Use the **Pen** tool for working through equations or diagrams, and the **Highlighter** to emphasise key steps.
- **Undo** individual strokes freely without losing context on earlier pages.
- Annotate over a previously prepared page to walk through a model answer.

## After the Session

1. On the tutor's device, open each page and choose **Export PNG** to save a clean image.
2. Share the PNG files with the student via any convenient method (email, shared folder, messaging app).
3. For a complete backup of the session, use **Export JSON** from the project settings.

## Why DrawShare Works for Tutoring

| Benefit | Detail |
|---------|--------|
| Multi-page organisation | Keep each topic or problem on its own named page |
| No distractions for the student | Read-only view with nothing to click or configure |
| Session stays private | No cloud, no recording, no third-party access |
| Works offline (single device) | Tutor can prepare pages in advance without a network |
| Lightweight export | PNG per page is easy to share and review later |
