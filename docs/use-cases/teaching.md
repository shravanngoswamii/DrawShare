# Teaching at the Whiteboard

Replace a physical whiteboard with a tablet and DrawShare. Your class sees every stroke as you draw it — on a projector at the front of the room, on their own laptops, or on a shared classroom display.

## Setup

**What you need:**

- Teacher device: iPad, Surface, or any tablet/laptop with a stylus or touchscreen
- Student/projector display: any device with a browser on the same Wi-Fi network

**Steps:**

1. Run DrawShare on the school's local network (or use the hosted version at your school's domain).
2. On the teacher device, open a project and navigate to your first lesson page.
3. Click **Share** — DrawShare shows a session code (e.g. `DRAW-7431`).
4. Display the session code on the projector or read it out.
5. Students open DrawShare and click **Join Session**, then enter the code.
6. Begin your lesson. Every stroke appears on student screens within milliseconds.

## During the Lesson

- Use **page switching** to move between prepared diagrams or blank working pages. Students follow automatically.
- **Undo** (`Ctrl+Z`) removes the last stroke without disrupting the flow.
- Use the **Highlighter** tool to draw attention to specific parts of a diagram.
- At the end of class, **Export PNG** to save each page for students to download as notes.

## Why DrawShare Works for Teaching

| Benefit | Detail |
|---------|--------|
| No student accounts | Students join with a code — nothing to register or install |
| Works on school networks | Local-only WebRTC; no firewall exceptions needed for external servers |
| Low latency | Strokes appear on viewer screens as fast as the local network allows |
| No distractions | Viewers see only the whiteboard — no chat, no notifications |
| Automatic save | Teacher's notes are saved to their device for later reference |
