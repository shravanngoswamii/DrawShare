# Remote Brainstorming

DrawShare turns any laptop into a shared visual thinking space for a room full of people. One person hosts the canvas and draws — everyone else watches the ideas take shape in real time.

## Setup

**What you need:**

- One host laptop with DrawShare open
- Any number of viewers on the same Wi-Fi (laptops, phones, tablets)
- Optional: a large display or TV connected to the host machine for a shared room view

**Steps:**

1. Host opens DrawShare, creates a new project (e.g. "Sprint Planning 2025-06").
2. Click **Share** and note the session code.
3. Participants open DrawShare and click **Join Session**, then enter the code.
4. Begin the session — all participants see the host's canvas live.

## During the Session

- Use **multiple pages** to separate branches of the idea tree. Name each page descriptively (e.g. "Risks", "Solutions", "Action Items").
- The host can switch pages mid-session; all viewers follow immediately.
- Use different colours to distinguish categories (e.g. red for blockers, green for ideas, blue for decisions).
- Sketch rough shapes and arrows to show relationships between concepts.

> **Coming soon:** Shape tools (rectangles, arrows, connectors) will make structured diagrams even easier to build.

## After the Session

1. Export each page as a **PNG** for a quick visual summary to share in Slack or email.
2. Export the full project as **JSON** to preserve the working session as an archivable file.

## Why DrawShare Works for Brainstorming

| Benefit | Detail |
|---------|--------|
| Immediate start | No account creation — share a code and you're live |
| Low barrier to join | Viewers only need a browser; no install, no sign-up |
| Multi-page branching | Separate idea clusters without losing context |
| Persistent session | Host's device saves everything; notes survive a reboot |
| Private by design | Ideas stay on the local network, not a third-party server |
