# Use Cases

DrawShare is a flexible tool that fits anywhere a whiteboard would — and some places a physical whiteboard cannot reach. Below is a collection of real-world scenarios with step-by-step setup guides.

## Scenarios

| Use Case | Summary |
|----------|---------|
| [Teaching](/use-cases/teaching) | Teacher draws on a tablet while the class follows along on their screens or a projector |
| [Tutoring](/use-cases/tutoring) | One-to-one session where the tutor walks a student through problems page by page |
| [Brainstorming](/use-cases/brainstorming) | Co-located team uses a single host canvas to capture ideas in real time |
| [Annotating Screenshots](/use-cases/annotation) | Paste or import an image and draw marks, arrows, and labels over it |
| [Personal Note-Taking](/use-cases/note-taking) | Offline digital notebook with multi-page projects and PNG export |

## Common Thread

Every scenario works with the same core properties of DrawShare:

- **No accounts required** — viewers join with a short code, nothing to sign up for.
- **Local-first** — sessions and notes stay on your device.
- **Cross-platform** — runs in any modern browser on Windows, macOS, Linux, iOS, or Android.
- **Zero configuration** — no server to manage unless you choose to self-host.
