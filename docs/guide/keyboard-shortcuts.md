# Keyboard Shortcuts

All shortcuts work when the canvas has focus (click the canvas area first if a shortcut is not responding).

## Tools

| Key | Action |
|-----|--------|
| `1` | Switch to Pen tool |
| `2` | Switch to Highlighter |
| `3` | Switch to Eraser |

## History

| Key | Action |
|-----|--------|
| `Ctrl` + `Z` / `Cmd` + `Z` | Undo last stroke |
| `Ctrl` + `Y` / `Cmd` + `Y` | Redo |
| `Ctrl` + `Shift` + `Z` / `Cmd` + `Shift` + `Z` | Redo (alternative) |

## Navigation & Interface

| Key | Action |
|-----|--------|
| `Escape` | Close open panel / cancel current action |
| `Arrow keys` | Scroll / pan the canvas |

## Tips

- On macOS, `Cmd` replaces `Ctrl` for all shortcuts.
- Undo/redo operate at the stroke level — one `Ctrl+Z` removes the last complete stroke.
- The `Escape` key dismisses the share panel, export dialog, or any open dropdown without committing changes.
